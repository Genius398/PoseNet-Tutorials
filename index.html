Lesson 1 - Introduction




PoseNet is a predefined JavaScript model which can be used to identify the x and y of 17 parts of the human body. It requiers webcam access. Don't worry! This is easy. Also, we will provide you with sample code to start of.
PoseNet models detect 17 different body parts or joints: eyes, ears, nose, shoulders, hips, elbows, knees, wrists, and ankles. Collectively these joints form a pose.
